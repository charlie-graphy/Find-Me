# Find-Me
<b>여러분의 열정과 취미를 찾아드립니다 </b><br>
<b>개발 언어 및 툴</b> : Html5, css, jQuery, JavaScript, MySQL  / Eclipse
<br>
<h3>설명 </h3>
-	희망적, 열정적, 자기개발/자기계발적, 이상적 분위기<br>
-	일상생활 속에서 자신의 행복과 열정, 꿈, 취미 등을 찾을 수 있도록 도와주는 웹<br>
-	자신이 가지고 있는 열정의 근원에 대해 생각해볼 수 있는 웹<br>

<h3>기능 </h3>
- MBTI 성격 유형 검사<br>
-자신에 대해 알아보기
